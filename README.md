\# Unity\_Game



\## 概要



UnityとC#を使用して制作している、2Dクォータービュー形式のゲームです。

魔法と猫をテーマにした世界を探索し、NPCとの会話やミニゲームを楽しめる作品を目指しています。



現在も開発を継続しています。



\## 開発環境



\* Unity 2022.3 LTS

\* C#

\* Visual Studio Code

\* Git / GitHub



\## 主な実装内容



\### フィールド探索



\* 8方向へのプレイヤー移動

\* 歩行状態と、ほうきに乗った状態の切り替え

\* 高低差のあるマップ

\* 坂道の移動処理

\* 高さとY座標を考慮した描画順の制御

\* 建物の外観と内装の切り替え



\### 会話・UI



\* NPCとの会話

\* 文字を順番に表示する会話演出

\* 複数ページの会話

\* 選択肢の表示

\* プレイヤー名の入力画面

\* HPやゲージの表示



\### ミニゲーム



\* 上方向へ進むアクションゲーム

\* 足場のリアルタイム生成

\* 移動床、落下床、シーソーなどのギミック

\* ミサイルや砲台による障害物

\* HPとアビリティゲージ

\* 勝敗判定と通常マップへの復帰処理



\## 操作方法



| 操作          | キー            |

| ----------- | ------------- |

| 移動          | 矢印キー          |

| 決定          | Enter / Z     |

| キャンセル       | Backspace / X |

| メニュー        | Escape / C    |

| 歩行・ほうきの切り替え | Space         |

| 選択肢の移動      | 上下キー / W・S    |
